This vscode extension adds colors to the maps of the wonderful game [Roguetype](https://github.com/Octachron/roguetype).

After installing, just open the file `game.ml` in a new editor and you should see the world types decorated as follows:

- free cells dimmed out
- obstacles, gates, door and altar on colorful backgrounds 
- monsters in various shades of red
- glowy pickable items

Happy typing
